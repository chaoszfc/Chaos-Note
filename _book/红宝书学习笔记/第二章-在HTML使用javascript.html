第二章

在HTML中使用JavaScript

HTML 4.01为script定义了下列6个属性

async：可选。表示应该立即下载脚本
charset:可选。表示通过src属性指定的代码的字符集
defer:可选。表示脚本可以延长到文档完全解析完和显示之后执行。只对外部脚本有效
src:可选。外部文件
type:可选。表示类型
